# 🏥 Barwaaqo Health – Somali Health Clinic & Wellness Center

**Barwaaqo Health** is a modern and professional healthcare website designed to provide accessible medical information, wellness services, and appointment booking for Somali communities. The platform focuses on improving healthcare accessibility through a clean, responsive, and user-friendly experience.

The website offers information about clinic services, specialist doctors, wellness programs, health education, and patient support in a professional and visually appealing interface.

---

## 📖 Project Overview

Barwaaqo Health is designed to create a trusted digital healthcare experience for Somali families and communities. The website aims to simplify access to healthcare services while promoting wellness, preventive care, and health awareness.

The project includes:

* Specialist doctor profiles
* Healthcare services information
* Appointment booking section
* Wellness center features
* Health education/blog section
* Responsive design for mobile and desktop users
* Professional medical UI/UX design

## 🚀 Installation & Setup

Follow these steps to run the project locally:

### 1. Clone the repository

```bash
git clone <https://github.com/ahmedhassan25-26/Future_FS_03.git >
```

### 2. Open the project folder

```bash
cd Barwaaqo-Health
```

### 3. Run the project

Open `index.html` in your browser.

You can also use **Live Server** in VS Code for a better development experience.
